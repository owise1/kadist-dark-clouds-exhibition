How to change wifi name

 
1. Open Terminal
2. ssh pi@kadist.local
3. password "raspberry"
4. sudo nano /etc/hostapd/hostapd.conf
5. change after "ssid="
6. exit nano ctrl-x
7. reboot - unplug or "sudo reboot"

How to pull latest version of exhibition

1. Open Terminal
2. ssh pi@kadist.local
3. password "raspberry"
4. ./kadist-dark-clouds/update-exhibition.sh
